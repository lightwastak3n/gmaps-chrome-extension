Google maps scraper

For each scan method you can choose a format: csv or Markdown, scroll duration 20-120 seconds. The default is 40 seconds which means extension will scroll each location given for 40 seconds or until it reaches bottom.

All the data scraped is public data. The extension scrapes visible data, so if the business's website isn't visible, it will not be scraped. The data scraped includes the name, type of business, rating, address, phone, website.

Scan current region

Zoom to the region of interest, do your search, choose format and time, and press Run. Extension will scroll for some time and give you the data for that region.

Find same business in multiple regions

Type the name of the business in the input field, type the regions in the text box below with one region per line like this:

madrid
warsaw
brent london

Press Run. The extension will scan same business type across all regions one by one and give you all the data as a single csv or markdown table.

Find different businesses in different regions

Don't type anything into a "type of business" field, type search terms into regions input field with one search per line. For example:

dentist in Paris
plumber in Las Vegas
lawyer in Berlin
